# 📘 Tema: Interfaces en Java

## 📌 Descripción

Este repositorio contiene una colección de actividades prácticas centradas en el uso de **interfaces en Java**, un concepto fundamental dentro de la programación orientada a objetos.

Las interfaces permiten definir **contratos** que las clases deben cumplir, garantizando que implementen ciertos métodos sin especificar cómo deben hacerlo.

---

## 🧠 ¿Qué son las interfaces?

Una **interfaz** en Java es una estructura que declara métodos (sin implementación, en la mayoría de los casos) que las clases deben implementar.

Se utilizan para:

* Definir comportamientos comunes entre distintas clases.
* Facilitar el **polimorfismo**.
* Desacoplar el código.
* Aplicar principios como el de **segregación de interfaces**.

---

## 🧩 Contenido del repositorio

En este repositorio encontrarás distintas actividades relacionadas con interfaces, como por ejemplo:

* Implementación de interfaces simples.
* Clases que implementan múltiples interfaces.
* Uso de interfaces para modelar situaciones reales (como dispositivos de oficina).
* Ejercicios prácticos para entender contratos y responsabilidades.

---

## 🎯 Objetivo

El objetivo de estas actividades es:

* Comprender cómo funcionan las interfaces en Java.
* Aprender a diseñar código más flexible y reutilizable.
* Aplicar buenas prácticas de programación orientada a objetos.

---

## 🚀 Conclusión

Las interfaces son una herramienta esencial en Java para construir aplicaciones bien estructuradas y mantenibles. Este repositorio reúne ejercicios diseñados para practicar y afianzar este concepto.
